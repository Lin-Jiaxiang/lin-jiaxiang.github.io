Hello SuperUI ​

介绍 ​

基于Vite3+TypeScript+antd的Vue3组件库开发框架 ​

在antd基础进行二次封装的业务组件 ​

快速开始 ​

仓库源在云效的制品仓库中，使用前需要把切换npm源，并登陆账号 ​
npm config set registry=https://packages.aliyun.com/642521a7a84870ae9d597435/npm/npm-registry/

npm login
  • 下载组件包
npm install @super/super-ui
  • 引入组件包
    1. 全局安装：在main.ts中加入
    import superui from '@super/super-ui'
import '@super/super-ui/es/style.css'
app.use(superui)
